# v0.2.9 — Codex Cost & Heatmap

## 主题

Codex tab 全面对齐 Claude tab —— 加上**本地 session JSONL 成本扫描** → 30 天 USD + per-model token → **消费热力图**（和 Claude 的 `UsageHeatmapView` 一致）。

- 新增 `CodexUsageCollector`（类比 `ClaudeUsageCollector`）：扫 OpenAI codex CLI 在 `~/.codex/sessions/**` 写的 rollout JSONL，提取 token / 估算 USD，喂进 v0.2.3 的 per-provider `UsageEventStore`。
- Codex tab 加消费热力图区块（复用 / 泛化 `UsageHeatmapView` + `UsageStatsService`）。
- 至此 Codex tab = 套餐徽章 + Session/Weekly 卡（含 pace + 趋势）+ 折线图 + 消费热力图 + Updated + 底栏 —— 与 Claude tab 形态一致。

**不含**：Codex CLI RPC / chatgpt.com Web 兜底数据源；Codex 阈值通知；Codex 多账号 —— 这些按需再排（spec §6 列）。
