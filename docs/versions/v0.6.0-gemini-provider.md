# v0.6.0 — Gemini Provider

## 主题

接入 Google 的 **Gemini Code Assist for Individuals**(个人 OAuth)作为第三条 provider,完整对标已有的 Claude / Codex 双段配额 + 历史折线 UI。本版本**不做**本机 session 离线统计(`~/.gemini/tmp` schema 不稳),推迟到 [gemini-cli #15292](https://github.com/google-gemini/gemini-cli/issues/15292) JSONL 提案落地后再做。

兑现 [ADR 0005](../adr/0005-reopen-multi-provider-direction.md) 中的 Gemini 占位 — 该 ADR 将 Gemini 列为"视用户需求再评估"的候选,issue [#27](https://github.com/methol/usage-bar/issues/27) 是用户需求落地。
