# v0.2.10 — Settings provider 列表 + 刷新纪律

## 主题

Settings 把「Primary Provider」单选下拉换成 **provider 列表**：可拖动排序、每个供应商一个启用/禁用开关、外加一个「是否在菜单栏显示」的单选子开关。顺手：

- 去掉 Settings 最底下的「Account」信息展示（Sign Out 仍在 popover 里）。
- Codex 也用与 Claude 同一个 polling interval 设置（不再固定 5 分钟）。
- **刷新纪律**：切 tab 和其它操作不再触发刷新；打开 popover 时刷一次（优先展示缓存）；刷新只剩两个入口——后台静默 timer、界面上的 Refresh 按钮。
- 菜单栏渲染 provider-aware（选 Codex 作菜单栏时显示 Codex 图标 + Codex 的窗口百分比）。
- `ProviderCoordinator` 统管 provider 顺序 / 启用集 / 菜单栏 provider / 非-Claude 的后台 timer。

**不含**：把 Claude 的后台 timer + 429 backoff 也收编进 `ProviderCoordinator`（迁移风险高，留后续）；Codex 作菜单栏时显示趋势箭头（需按 provider 选对应历史服务，独立的活）；给 Codex 做专属菜单栏 logo 图片（先用 SF Symbol）。
